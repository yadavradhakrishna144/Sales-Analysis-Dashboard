# Sales Analysis Dashboard

## Project Overview

This project analyzes retail sales data to identify trends, profitable categories, and regional performance.

The project demonstrates the complete data analysis workflow including data cleaning, analysis, and dashboard creation.

## Tools Used

* Python (Data Cleaning & EDA)
* Excel (Data Analysis)
* Power BI (Interactive Dashboard)

## Key Insights

* The West region generated the highest sales.
* Technology category produced the highest profit.
* Sales show strong growth in the last months.

* Dataset
* Python analysis
